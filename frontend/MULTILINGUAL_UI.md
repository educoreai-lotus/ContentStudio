# Multilingual UI - Frontend Implementation

## Overview

The Multilingual UI allows trainers and learners to:
1. Select preferred language
2. View lesson content in selected language
3. See language statistics and popularity
4. Understand content source (cache/translation/generation)

## Components Created

### 1. LanguageSelector Component
**Location:** `frontend/src/components/LanguageSelector.jsx`

**Features:**
- Dropdown for language selection
- Shows language flag and name
- Displays "Cached" badge for frequent languages
- Shows language statistics (optional)
- Theme-aware styling

**Usage:**
```jsx
<LanguageSelector
  selectedLanguage={selectedLanguage}
  onLanguageChange={setSelectedLanguage}
  theme={theme}
  showStats={true}
/>
```

### 2. LanguageStatsDashboard Component
**Location:** `frontend/src/components/LanguageStatsDashboard.jsx`

**Features:**
- Summary cards (Total Languages, Frequent, Non-Frequent)
- Frequent languages list with request counts
- Popular languages ranking
- Visual indicators for cached languages
- Loading and error states

### 3. LessonViewWithLanguage Component
**Location:** `frontend/src/pages/Lessons/LessonViewWithLanguage.jsx`

**Features:**
- Language switcher in header
- Content source indicators (Cache/Translation/Generation)
- Displays content in selected language
- Shows translation source language
- Handles missing content gracefully

**Route:** `/lessons/:topicId/language`

### 4. LanguageStatsPage Component
**Location:** `frontend/src/pages/Multilingual/LanguageStatsPage.jsx`

**Features:**
- Full-page language statistics dashboard
- Comprehensive language usage analytics

**Route:** `/languages/stats`

## Services

### multilingual.js
**Location:** `frontend/src/services/multilingual.js`

**Methods:**
- `getLessonByLanguage(lessonId, preferredLanguage, contentType)` - Get lesson in preferred language
- `getLanguageStats()` - Get all language statistics
- `getLanguageStat(languageCode)` - Get specific language statistics

## Routes Added

```jsx
<Route path="/lessons/:topicId/language" element={<LessonViewWithLanguage />} />
<Route path="/languages/stats" element={<LanguageStatsPage />} />
```

## Content Source Indicators

### Cache (Green Badge)
- Content stored in Supabase Storage
- Fast retrieval
- Frequent languages (en, he, ar)

### Translation (Blue Badge)
- Content translated from another language
- Shows source language
- Generated on-the-fly

### Generation (Purple Badge)
- Content generated by AI
- No source content available
- Full AI generation
